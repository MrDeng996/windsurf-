# windsurf-
Human Approval 是一个为 AI 编码助手设计的 windsurf 扩展，它建立了一套优雅的暂停与确认机制。当 AI 完成一项任务或需要人类指导时，它会调用一个特殊的工具，暂停执行并弹出一个现代化的交互界面，等待您的下一步指令。  这不仅仅是一个暂停按钮，更是一个功能强大的人机交互中心


## 核心功能

- **Morden UI：提供一个美观且功能丰富的界面，用于审查 AI 的工作和提供反馈。
- **MCP 协议兼容**：通过标准的模型上下文协议（MCP）与 AI 通信，支持 HTTP 和 stdio 模式。
- **指令队列**：允许您输入多条指令，然后一次性批量发送给 AI。
- **文件上传**：支持拖放或选择文件（图片、文档等）并将其发送给 AI。
- **快捷指令**：可自定义和预设常用指令，一键发送，提高效率。
- **规则生成**：通过命令一键在工作区创建 `.windsurfrules` 文件，指导 AI 如何使用此工具。
